# 한국의 건강지도 — 프로젝트 문서

> 건강검진통계연보(2015-2024) 기반 인터랙티브 대시보드
>
> Live: https://drmoony.github.io/k-healthmap

---

## 1. 프로젝트 개요

건강검진통계연보 10개년치(2015-2024) 데이터를 파싱하여, 제약회사의 대사질환 시장 접근/정책 분석에 활용 가능한 단일 HTML 인터랙티브 대시보드를 구축한 프로젝트.

- **최종 산출물**: `index.html` (200KB, 자체 완결형 — 서버 불필요)
- **데이터 범위**: 18개 검진항목 × 17시도 × 15연령대 × 3성별 + 10년 추이
- **디자인 컨셉**: 수묵화/한지 톤 (Noto Serif KR, ink-wash 색상 팔레트)
- **차트 라이브러리**: Chart.js 4.4.1 (CDN)
- **지도**: SVG 코로플레스 (Natural Earth 10m → geopandas → SVG path)

### 단일 HTML 아키텍처를 선택한 이유

1. GitHub Pages에서 별도 빌드 없이 바로 배포 가능
2. 오프라인 환경에서도 파일 하나만 열면 동작 (CDN 제외)
3. 제약회사 내부 공유 시 첨부파일 하나로 전달 가능
4. 데이터가 JS 변수로 내장되어 별도 API/DB 서버 불필요

## 3. 탭 구조 및 기능

### 3.1 종합현황

| 요소 | 설명 |
|---|---|
| KPI 카드 ×4 | 일반검진 수검률, 암검진 수검률, 비만율, 대사증후군 위험군 |
| KPI 카운팅 | 페이지 로드 시 0→목표값 easeOutCubic 애니메이션 (1.5초) |
| KPI 클릭 | 수검률 카드 → 수검현황 탭, 비만율/대사증후군 → 대사질환 탭 이동 |
| 10년 추세 차트 | 4개 지표 라인 차트 (2015-2024) |
| 시도별 순위 | 비만율/대사증후군 수평 바 차트 + 전국 평균 레퍼런스 라인 |
| 국제비교 | Chart.js 수평 바, BMI 기준 차이(≥25 vs ≥30) 명시 |

| 요소 | 설명 |
|---|---|
| SVG 코로플레스 지도 ×2 | 비만율 (노랑→빨강), 대사증후군 (연두→남보라) |
| 연도 슬라이더 | 2015-2024 드래그로 지도 색상 실시간 변화 |

| 시도별 비교 바 | 전국 평균 기준 빨강(초과)/녹색(미만) 색상 분리 |

## 5. Chart.js 커스텀 플러그인

### 5.1 dataLabelPlugin (`inkLabels`)

스택형 바 차트의 각 세그먼트에 수치 표시.

- 바 높이 ≥ 18px & 값 ≥ 5%: 세그먼트 **내부**에 표시
- 바가 작지만 비정상(상위 절반 카테고리) & 값 ≥ 0.5%: **외부** 빨간 필 라벨 + 연결선
- 글씨색: 배경 밝기(ITU-R BT.709 기준 luminance > 140) → 검정, 아니면 → 흰색

### 5.2 crosshairPlugin

마우스오버 시 십자 가이드라인 + 축 라벨 필 + 데이터 포인트 수치 버블.

- 세로선 + 가로선 (점선, 반투명 빨강)
- Y축: 카테고리명 또는 수치 → 빨간 둥근사각 배경 필
- X축: 동일
- 가장 가까운 데이터 포인트에 수치 버블 (검정 배경)

### 5.3 refLinePlugin

수평/수직 레퍼런스 라인 (전국 평균, OECD 평균 등).

- 점선 스타일
- 차트 영역 **내부**에 흰색 배경 라벨 (잘림 방지)

## 6. 데이터 파이프라인

```
원본 엑셀 (건강검진통계연보)
    │
    ▼  openpyxl (Python)
제6편 (검진결과별) ──→ FULL_DATA (18항목 × 시도/연령 × 성별)
제5편 (문진)       ──→ LIFESTYLE_DATA (흡연/음주/운동)
제7편 (대사질환)   ──→ MET_PROV (시도별 10년 추이)
    │
    ▼  build_dashboard.py
중간 JSON (data_bundle.json)
    │
    ▼  build_v5.py + build_v5_js.py
최종 HTML 조립
    │
    ▼  geopandas + Natural Earth 10m
KOREA_PATHS (SVG path 데이터)
```

### 엑셀 파싱 핵심 로직

건강검진통계연보 엑셀은 merged cell 구조:

```
서        울  |  합계  |  S.T  |  값1  |  값2  |  ...
Seoul        |  남자  |  M    |  값1  |  값2  |  ...
             |  여자  |  F    |  값1  |  값2  |  ...
부        산  |  합계  |  S.T  |  ...
```

- 시도명: column 1, 한글 공백 제거 (`re.sub(r'\s+','',v)`)로 정규화
- 성별: column 2/3 (`합계/남자/여자` or `Total/M/F` or `S.T`)
- 값: column 4~ (절대수 → 비율로 변환: `value / total × 100`)

## 7. 디자인 시스템

### 색상 팔레트 (수묵화 테마)

| 변수 | 값 | 용도 |
|---|---|---|
| `--bg` | `#f5f0e8` | 배경 (한지) |

| `--ink` | `#1a1a1a` | 주요 텍스트 (먹색) |
| `--accent` | `#8b1a1a` | 경고/강조 (주홍) |
| `--gold` | `#b8860b` | 보조 강조 (금박) |
| `--jade` | `#2d6a4f` | 긍정 지표 (옥색) |

### 지도 색상 보간

| 지도 | 범위 | 색상 |
|---|---|---|
| 비만율 | 30% → 45% | 노랑 → 주황 → 진빨강 |
| 대사증후군 | 60% → 78% | 연두 → 청록 → 네이비 → 보라 |

### 애니메이션

- `fadeInUp`: 카드 등장 (0→20px, 0.5s ease-out)
- `countUp`: KPI 숫자 카운팅 (0→목표값, 1.5s easeOutCubic)
- 카드 호버: `translateY(-4px) scale(1.01)` + 그림자 확대
- 스택형 바: 데이터 변경 시 Chart.js 기본 전환 애니메이션

## 10. 기술적 문제 해결 기록

| 문제 | 원인 | 해결 |
|---|---|---|
| Chart.js y축 무한 확장 | `responsive:true` + container에 `overflow:hidden` 없음 | `.ch{overflow:hidden}` + canvas `position:absolute` |
| 지도 원형 블롭 | 수작업 SVG 좌표가 원에 가까움 | Natural Earth 10m + geopandas로 실제 지형 변환 |
| 여성 데이터 누락 | 파서가 여자(F) 행 스킵 | 성별 감지 로직 수정 (column 2/3 기반) |
| Python f-string `{{}}` 오염 | f-string에서 JS 중괄호 이스케이프가 JSON 데이터 파괴 | Template substitution 방식으로 전환 |
| 두 script 블록 충돌 | const 재선언 에러 | 단일 script 블록으로 병합 |
| `roundRect` 미지원 | Chrome 99 미만 브라우저 | 폴리필 추가 |
